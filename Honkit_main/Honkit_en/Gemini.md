# Gemini
ailia DX insight can also perform text generation using Gemini. For details on Gemini, please refer to [here](https://ai.google.dev/gemini-api/docs/models/gemini?hl=ja).

## How to Connect to Gemini
### Calling the Settings Window
1. In the initial screen of ailia DX insight, click the gear icon in the upper right to display the settings window.<br>
![setup_03.png](/img/setup_03.png)<br>
1. Click "Gemini" in the "Chat AI" items and select "+ Add".
![gemini_01.png](/img/gemini_01.png)<br>

### Register Gemini Settings
1. A window will open to register ChatGPT deployed in Gemini. Fill in each item.<br>
![gemini_02.png](/img/gemini_02.png)<br>
  * Name: Name used for UI display
  * Description: Use as a memo as needed
  * Model: Deployed model name
  * Maximum Token Length: Maximum token count of the model

2. When "Register API Key" is displayed in the API key section, set it.<br>
For information on API keys and model names, please refer to [here](https://ai.google.dev/gemini-api/docs/api-key?hl=ja).
